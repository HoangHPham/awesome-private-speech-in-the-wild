# 📌 Awesome Speech Privacy in the Wild

A curated collection of resources on **privacy-preserving speech technology in real-world conditions**, including noisy environments, overlapping speakers, and multi-device scenarios.

This repository aims to support research in robust and trustworthy speech systems that preserve user privacy without sacrificing performance.
